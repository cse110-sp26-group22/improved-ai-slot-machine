# AI Use Plan:

## Overview
My overall plan is to first let the AI read through our prompt.md, and let it generate the first prototype, then check if there are any bug in functionality part. After making sure that the functionalities are working as intended, I will proceed to ask AI to improve visual componenets.

## Model
We decided to use Gemini-3-flash model.

## Breakdown
My aim is to break the task down into three different stages of refinement and generation with the AI
1. Prototyping: Use AI to generate the prototype just by giving the prompt.md
2. Debugging: Test the functionalities and debug.
3. Refinement: Improve the visual and animations.
