# AI Use Plan:

## Overview
Research summary and evaluation rubric will be used to construct initial prompt for slot machine program. Further refinement will be done using evaluation rubric and human review. Refinement will consist of multiple steps (> 3) and will continue until program satisfies rubric conditions within a reasonable degree.

## Model
I will be using Gemini 3 Flash for my initial prompting and refinement. Gemini CLI is free which reduces costs, while providing access to general "flash" models and lightweight "lite" models. Under a free account, token availablity is shared between Gemini 2.5 Flash and Gemini 3 Flash, so Gemini 3 Flash was chosen for its superior reasoning and code generative capabilities.

## Breakdown
First step will consist of the initial prompt with consecutive refinement steps pertaining to best improvements that can reasonably be done at each step. Prompt given at each step will be recorded alongside a general description of the results of the previous prompt. 

1. Research review
2. Initial prompt design 
3. Run first prompt
4. Program Evaluation 
5. Refinement prompt design
6. Run Refinement prompt
7. Repeat steps (4-6) until program satsisfies rubric
8. Conclusion
