# AI Use Plan:

## Overview
My overall plan is to guide the AI through the development process by first starting it with a list of criteria from our rubric that needed to be satisfied. I will then analyze the code and the final product to find areas that could be improved upon and then use a new prompt to target that lacking area.

## Model
We decided on using the Flash version of Gemini-3, primarily because it is free for use. However, Gemini's code helper is generally considered to be behind models like Claude in terms of raw code assistance capabilities. By working with Gemini, my personal interest was in finding out if Gemini is still at a competitive level to paid options like Claude Code

## Breakdown
My aim is to break the task down into two different stages of refinement and generation with the AI
1. Prototyping : what does the AI output given just the outline of the project
2. Cleaning up functional and visual problem areas: Refining what makes the slot machine a slot machine
